💻 IMPLÉMENTATION TECHNIQUE
═══════════════════════════════════════════════════════════════════════════════

Stack i18n:
  ✅ Context API React (zéro dépendances externes)
  ✅ localStorage pour persistance
  ✅ Hook useLanguage() pour accès facile

  LanguageProvider (root layout)

  useLanguage() hook

  ✅ Pas de chargement async
